# detourguru.github.io

프론트엔드 개발자 치섭의 기술 블로그입니다.
프로젝트에 투입되어 아키텍처 설계부터 운영까지 직접 부딪히며
풀어온 문제들을 기록합니다.

## 다루는 주제

- React/TypeScript 기반 상태 관리, 렌더링 최적화
- 1인 개발 환경에서의 AI 협업 워크플로
- 버그 복기 — 무엇이 왜 깨졌고, 어떻게 고쳤는지
- 측정 기반 의사결정 (Profiler, 실측 데이터)
